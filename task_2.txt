3. Creating dump of example-database:

(base) annaniukkanen@Annas-Air MySQL % mysqldump example 


(base) annaniukkanen@Annas-Air MySQL % mysqldump example > example.sql


Creating a new database sample:

mysql> show databases;

(base) annaniukkanen@Annas-Air MySQL % mysql sample < example.sql
